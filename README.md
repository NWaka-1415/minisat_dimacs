# minisat_dimacs
某H大学の某情報基礎学Bの塗り分け問題をdimacs形式にエンコードを実現するPythonプログラム。今後情報基礎学Bを取る人はぜひ使ってね！

## 使い方
プログラムの以下の箇所(main内)に塗り分ける領域数，つながっている領域の組み合わせ（タプル）のリスト，塗る色の数を記述して実行するとコンソールにエンコード結果が表示されます。

```
gcp = GCP(4, [(1, 2), (1, 3), (1, 4), (2, 4), (3, 4)], 3)
```
